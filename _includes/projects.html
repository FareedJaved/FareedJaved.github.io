Bands In Town

An application written using Node.js which makes use of the BandsInTown Api to see if Chance The Rapper will be performing near me. The application also uses the Twilio Api to send a notification to me as well.

EarthQuake Map

Project from a Coursera class sponsored by the University of California, San Diego. Learned event driven and Object Oriented Programming while creating a interactive map that visualizes earthquakes world wide.

No On Question D
