Bands In Town

An application written with Node js which utilizes the BandsInTown Api to see if a specific artist will be performing near the user. The application also uses the Twilio Api to send a notification to the user as well.

EarthQuake Map

Project from a class on Coursera.com sponsored by the University of California, San Diego. Learned event driven and Object Oriented Programming while creating a interactive map that visualizes earthquakes world wide.

No on Question D

Worked with Progressive Maryland to develop a website that increased awareness for a ballot question that was voted on during the 2016 election.
