Bands In Town

An application written with JRuby which utilizes the BandsInTown Api to see if an artist will be performing near the user. The application also uses the Twilio api to send a notification to the user as well.

EarthQuake Map

Project from a Coursera class provided by the University of California, San Diego. Learned event driven and Object Oriented Programming while creating a interactive map that visualizes earthquakes around the world.

Six Degrees Of Kevin Bacon

Utilized graph data structure to find an actor's Kevin Bacon Number. Studied the Breadth-First traversal algorithm.
